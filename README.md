## Descripción del taller 

Este taller de programación en Python fue desarrollado en equipo con el objetivo de fortalecer nuestras habilidades en lógica de programación y resolución de problemas. Durante el trabajo, colaboramos en la implementación de distintos ejercicios que incluyen el ingreso y procesamiento de datos, cálculo de promedios, identificación de valores máximos y mínimos, búsqueda de elementos, ordenamiento de listas y generación de combinaciones. Además, analizamos la eficiencia de cada algoritmo mediante la complejidad Big O. El trabajo en grupo permitió intercambiar ideas, corregir errores y mejorar nuestras soluciones de manera conjunta.

## Contenido del repositorio: 

-Ejercicio 1:
Pide 3 números al usuario y calcula el promedio sumándolos y dividiéndolos para mostrar el resultado.

-Ejercicio 2:
Solicita varios números, los guarda y determina cuál es el mayor y cuál es el menor.

-Ejercicio 3:
Permite ingresar varios números y luego busca uno específico recorriendo la lista para indicar si existe y su posición.

-Ejercicio 4:
Recibe una lista de números y la ordena de menor a mayor usando el método burbuja, mostrando antes y después.

-Ejercicio 5:
Divide un número entre 2 repetidamente hasta llegar a 1 o menos, contando cuántas divisiones se realizan.

-Ejercicio 6:
Genera todas las combinaciones posibles de pares entre los números ingresados.
